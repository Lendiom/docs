## Description

The **Loan** variable contains all of the information regarding a loan.

## Data

### Variables
* `Label`: the visible name of the loan
* `Status`: the status of the loan: `active`, `inactive`, `late`, `in-default`, `draft`, `paid-off`

### Functions
None so far.
